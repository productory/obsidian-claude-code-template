Quick daily status capture that appends to the current weekly file.

## Steps

1. **Load active context**: Read `20-work/_active-context.md` to load working memory — current priorities and open blockers.

2. **Find current weekly file**: Locate the latest active weekly file in `20-work/20.02-weekly/`.

3. **Determine today**: Identify today's day of the week (Mon-Fri).

4. **Ask for updates**: Prompt the user for:
   - **Done**: What did you accomplish today?
   - **Blocked**: Anything blocking your progress?
   - **Next**: What's planned for tomorrow?

5. **Append to daily log**: Add today's entries under the appropriate day in the Daily Log section of the weekly file.

6. **Check 1Thing progress**: Briefly ask if progress was made on the week's 1Thing. Note it if yes.

7. **Update active context**: Update `20-work/_active-context.md` — update blockers if changed, append a brief session note with today's key update.

8. **Quick summary**: Confirm what was recorded. Keep this interaction fast and lightweight — this is not a full review.

Adopt the **Chief of Staff** agent persona. Keep it brief and action-oriented.
