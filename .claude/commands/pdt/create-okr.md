Initialize a yearly OKR file or update an existing one.

## Steps

1. **Check for existing OKR**: Search `20-work/20.01-okr/` (or specified workspace) for a current year OKR file.

2. **If no OKR exists — Create new**:
   - Read the template from `00-system/00.01-templates/tpl-okr-yearly.md`
   - Ask the user for their yearly vision
   - Walk through Q1 objectives and key results (start with current quarter)
   - Create the file as `YYYY-okr.md` in the OKR folder

3. **If OKR exists — Update**:
   - Read the current OKR file
   - Ask what the user wants to update:
     - Add/modify objectives or key results
     - Update progress on existing key results
     - Plan next quarter's priorities
   - Make the requested changes

4. **Monthly priorities**: For the current quarter, help break down into monthly priorities if not already done.

5. **Confirm**: Show the updated OKR and summarize changes.
