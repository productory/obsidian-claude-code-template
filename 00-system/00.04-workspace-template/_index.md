# Workspace Template

This is the clonable workspace template. To create a new workspace:

1. Copy this folder
2. Rename `XX` prefix to the next available number (e.g., `30`)
3. Rename subfolder prefixes accordingly (e.g., `XX.01` → `30.01`)
4. Add workspace entry to `CLAUDE.md`

## Structure

- **_active-context.md** — Claude's working memory for this workspace (maintained automatically across sessions)
- **XX.01-okr/** — Yearly OKR with quarterly/monthly breakdown
- **XX.02-weekly/** — Weekly 1Thing plans and reviews
- **XX.03-meetings/** — Meeting notes linked to projects and weeklies
- **XX.04-projects/** — Lightweight project management with PIC
- **XX.05-archive/** — Completed/inactive items
